CSS Selector Examples

This page is for experimenting with the application of various CSS Selector techniques

Apply and experiment with each of these advanced CSS selectors:

universal selectors
* ALL elements match the univeresal selector
descendant selectors
selector01 selector02 all selector02s that descend from selector01s are styled. separate the descending selectors with a spacebar space
child selectors
selector01 > selector02 style all selector02 that are children of selector01. separate the child selectors with a greater than sign
pseudo-class selectors
selector:link, selector:visited, selector:hover, selector:active style an element based upon it's state. assign the pseudoclass using the colon
selector:first-line, selector:first-letter, etc style the first letter or line of the selector
selector:nth-child(n) select the (n)th child
selector:nth-child(2n) select every other child
selector:nth-child(2n+1) select every other child+1
attribute selectors
selector[attribute=value] only style selectors that include the attribute or the attribute and value. use square brackets to define attribute selectors
  • selector[attribute] style selector where the specified attribute is present
  • selector[attribute="value"] only those where attribute is assigned 'value'
  • selector[attribute^="va"]where attribute value BEGINS with 'va'
  • selector[attribute$="ue"]where attribute value ENDS with 'ue'
  • selector[attribute*="alu"]where attribute value CONTAINS 'alu'
